\subsubsection{Unreachable code}
Currently permitted (but could easily be outlawed):

\begin{python}
def foo(x):
  return x + 1
  return y    # y undefined, but ok since this code is unreachable
\end{python}

\subsubsection{Conditionals without \code{else} clauses}

An \code{if} statement without an \code{else} clause is equivalent to an \code{if}
statement with an \code{else} clause whose body is a $\exPass$ statement.

\subsubsection{Local assignment in conditional clauses}

We would like the ``inner'' (shadowing) assignment of \code{x} to be permitted and remain local to that
branch of the conditional:

\begin{python}
def foo():
  x = 6
  if b:
    x = 3
    y = x + 1
  else:
    y = 0
  return x  # needs to be "lexically scoped", but which x is being referred to?
\end{python}
